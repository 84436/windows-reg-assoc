"Open in VSCode" context menu

NOTE
----
Exported keys are not human-readable due to REG_EXPAND_SZ values being encoded as a string of hexadecimals.

REFERNCES
---------
%V                 = Windows Explorer variable
                     "Current directory" if loosely understood in this context.
                     https://web.archive.org/web/20111002101214/http://msdn.microsoft.com/en-us/library/windows/desktop/cc144101(v=vs.85).aspx#verbs
&                  = underlined letter (for keyboard navigation)
"Background\shell" = "background" context menu of the current directory
"Shell"            = context menu on a directory

KEYS: Format: [Location] Name, Type, Data
-----------------------------------------

[HKCR\Directory\Background\shell\VSCode]
@       REG_EXPAND_SZ   Open w&ith Code
Icon    REG_EXPAND_SZ   <Path to VSCode>

[HKCR\Directory\Background\shell\VSCode\command]
@       REG_EXPAND_SZ   <Path to VSCode> "%V"

[HKCR\Directory\Shell\VSCode]
@       REG_EXPAND_SZ   Open w&ith Code
Icon    REG_EXPAND_SZ   <Path to VSCode>

[HKCR\Directory\Background\shell\VSCode\command]
@       REG_EXPAND_SZ   <Path to VSCode> "%V"
